Nocalhost is a new way of Cloud Development

Localhost Dev Mode

With Localhost dev mode, developers have to run all services in DevMode locally. Everyone coding in isolated.

  • Limited by local resources, hard to develop large applications
  • Difficult to setup and maintain the development environment
  • Difficult in integrated debugging
  • Lots of differences compared to production environment
  • Limited by office network, can't work flexibly
Nocalhost Dev Mode

With Nocalhost dev mode, all services run on cloud. Developers can switch services to DevMode and connect IDE to the service containers by sharing same environment.

  • Flexible cloud resources to easily handle large application development
  • Dev and test environment provision on demand, convenient, cheap and fast
  • Multi-developer shared space make debugging collaboration easy
  • Almost same with production environment
  • Coding anytime anywhere
