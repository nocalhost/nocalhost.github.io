What is Nocalhost?

It's a

Bridge

Between Cloud And IDE

Smoothly, code changes be transfered from IDE to cloud without commit, push or pull.

It's a

Space

To Share Dev Environment

Developers can share the same development environment and conduct joint debugging.

It's a

Extension

Of Kubernetes

Amazing enhancement of Kubernetes and Helm, developers codes on Kubernetes with on friction.

It's a

Spotlight

To Explore MicroService

By using Nocalhost, developers can focus on a certain service in the microservices, switch to development mode and debug easily.
